# 🏥 비콘 검수 대시보드

**일산병원 현장 비콘-자산 매칭 검수 시스템**  
단일 HTML 파일 하나로 동작하는 오프라인 검수 도구.  
서버 없이 브라우저에서 직접 실행, 검수 결과는 localStorage에 영구 저장.

---

├── README.md                      ← 이 문서

## 🖥️ 화면 구성

### 상단 헤더
| 요소 | 설명 |
|------|------|
| 검수 진행률 바 | 병동명 + 완료/전체/% 실시간 표시 |
| 📋 검수 결과 | 일자별 히스토리 탭 |
| 📂 엑셀 업로드 | 새 자산 데이터로 교체 |
| 📥 내보내기 | 검수 결과 CSV 다운로드 |

### 좌측 사이드바
- **전체 현황 홈** — 홈 화면 복귀
- **병동 목록** (21개, 숫자 오름차순)
- **부서 목록** (94개, 가나다순)

### 홈 화면
- KPI 카드 4개: 전체 자산 / 병동 자산 / 검수 완료 / 부서 자산
- **🏥 병동 검수 현황** 도넛 차트 (2,516개)
- **🏢 부서 검수 현황** 도넛 차트 (2,515개)
- 카테고리 분포 상위 10 바차트
- 병동별 / 부서별 자산 현황 카드 그리드 (실시간 검색 + 클릭 이동)

### 검수 화면 컬럼 (8개)
```
No. | 자산명 | 자산코드 | 카테고리 | 비콘 | 자산별칭 | 검수결과 | 메모
```
필터 탭: **전체 / 미완료 / ✓ 일치 / ✗ 불일치 / 비콘없음**

### 📋 검수 결과 히스토리 탭
- 전체 화면 오버레이 (`← 홈으로` 버튼으로 복귀)
- **일자별 그룹핑** (접기/펼치기, 날짜 내림차순)
- **컬럼별 필터링**: 병동/부서, 자산명, 자산코드, 자산별칭, 비콘번호
- **컬럼별 정렬**: 헤더 클릭 → ▲▼ 오름/내림차순
- **컬럼 너비 조절**: 경계 드래그 + 더블클릭 auto-fit
- 날짜 범위 필터 + CSV 내보내기 + 전체 초기화
- 통계 카드: 전체 검수 / 일치 / 불일치 / 검수 일수

## 📊 테이블 기능

### 컬럼 너비 조절
- 헤더 오른쪽 **경계 드래그** → 폭 조절 (`col-resize` 커서)
- 경계 **더블클릭** → 실제 DOM scrollWidth 기준 자동 조절
- 폭 조절 ↔ 정렬 클릭 완전 분리

### 컬럼 순서 변경 (DnD)
- 헤더 **드래그 앤 드롭** → 전체 컬럼 이동
- 드래그 중 고스트 미리보기 (헤더 + 첫 5행 데이터)
- `window.COL_ORDER` 배열로 순서 관리

### 커서 구분
- 헤더 텍스트 → `pointer` (정렬)
- 오른쪽 경계 → `col-resize` (폭 조절)

## 🔍 검색 기능

### 병동/부서 그리드 검색
- 홈 화면 각 섹션 상단 검색창
- 실시간 필터링 + 매칭 글자 노란색 하이라이트
- × 버튼 초기화 / 결과 없음 메시지

### 히스토리 컬럼 필터
- 병동/부서, 자산명, 자산코드, 자산별칭, 비콘번호 입력창
- 여러 컬럼 동시 필터링 (AND 조건)
- 필터 적용 시 날짜 블록 카운트 실시간 갱신

## 💾 localStorage 영구 저장

| 키 | 내용 |
|----|------|
| `beacon_res_v3` | 검수 결과 (자산별 OK/NG/메모/타임스탬프/병동) |
| `beacon_raw_v3` | 엑셀 업로드 후 교체된 자산 데이터 |

**저장 시점**: ✓/✗ 클릭 즉시 / 메모 입력 시 / 엑셀 업로드 완료 시

> 같은 브라우저 + 같은 파일 경로에서만 유지. 주기적으로 CSV 백업 권장.
